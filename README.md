# Changelog: `1.1` -> `1.2`

## Added

- Добавлена кнопка **Гибкая настройка** во вкладке **Профиль → Тесты**.
- Добавлена гибкая настройка поиска тестов:
  - выбор конкретного предмета;
  - выбор конкретной недели;
  - возможность сканировать всё сразу, как в старой версии.
- Добавлена настройка **Автоматическая подсветка**.
- Добавлена автоподсветка тестов на странице журнала `https://interneturok.ru/h/journal/` без необходимости каждый раз открывать popup и нажимать кнопку **Подсветить тесты**.
- Добавлено отображение количества вопросов в тесте.
- При подсветке тестов возле `ДЗ` теперь показывается число вопросов, например: `ДЗ 10`.
- Добавлена цветовая подсказка для заданий на сопоставление/перетаскивание:
  - поле вопроса подсвечивается одним цветом;
  - правильный ответ для него подсвечивается тем же цветом;
  - каждая следующая пара получает другой цвет.
- Добавлены новые сохранённые настройки для гибкой настройки тестов.

## Changed

- Улучшена логика поиска тестов в журнале: теперь можно ограничивать поиск по предмету и/или неделе.
- Улучшена подсветка найденных тестов: теперь она может дополнительно показывать количество вопросов рядом с `ДЗ`.
- Изменена подсказка для match/drag-заданий: используется цветовое сопоставление пар.
- Автоподсветка стала работать из content script, поэтому может запускаться на странице журнала без открытия popup.
- Обновлена логика хранения и применения настроек тестов.
- Обновлён `manifest.json`:
  - `version`: `1.1` -> `1.2`.

# Что делает каждая функция и как ей пользоваться

Ниже описание функций расширения простыми словами. Разделы идут так же, как основные вкладки popup: **Имя**, **Копия**, **Оценки**, **Ответы**, **Профиль**, а потом настройки и служебные функции.

---

## 1. Вкладка «Имя»

### 1.1 Просмотр активной вкладки

**Что делает:**

Показывает, с какой страницей сейчас работает расширение. Это нужно, чтобы понимать, куда будут отправляться команды.

**Как пользоваться:**

1. Открой нужную страницу.
2. Открой popup расширения.
3. Во вкладке **Имя** посмотри блок **Активная вкладка**.

**Важно:**

Если открыта системная страница Chrome, расширение может не работать, потому что Chrome запрещает внедрять туда content script.

---

### 1.2 Просмотр ключей из URL

**Что делает:**

Показывает важные параметры из адреса страницы, например `itemId`, `lessonId`, `quarter` и другие параметры, если они есть в ссылке.

**Как пользоваться:**

Обычно ничего делать не нужно. Это служебная информация для отладки. Если не знаешь зачем это нужно — лучше не трогать.

---

### 1.3 Получение текущего имени пользователя

**Что делает:**

Пытается прочитать текущее имя пользователя на странице Jitsi через внутреннее состояние страницы.

**Как пользоваться:**

1. Открой страницу Jitsi.
2. Открой popup расширения.
3. Во вкладке **Имя** появится текущее имя.

**Важно:**

Для чтения имени используется `injected.js`, потому что обычный content script не имеет прямого доступа к `window.APP` страницы.

---

### 1.4 Смена имени пользователя

**Что делает:**

Меняет display name пользователя на странице Jitsi.

**Как пользоваться:**

1. Открой страницу Jitsi.
2. Открой popup расширения.
3. Во вкладке **Имя** в поле **Новое имя** введи нужное имя.
4. Нажми **Применить имя**.

**Что происходит внутри:**

Расширение отправляет команду на страницу, а `injected.js` пытается изменить имя через Jitsi API: `conference.setDisplayName` и dispatch в store.

---

### 1.5 Обновление связи со страницей

**Что делает:**

Повторно внедряет content script на активную вкладку, если popup потерял связь со страницей.

**Как пользоваться:**

1. Если имя не читается или команда не срабатывает, нажми **Обновить связь**.
2. После этого попробуй снова выполнить нужное действие.

---

### 1.6 История имён

**Что делает:**

Сохраняет ранее использованные имена, чтобы их можно было быстро выбрать снова.

**Как пользоваться:**

1. Один раз примени имя.
2. Оно появится в истории.
3. В следующий раз можно нажать на имя из истории и использовать его повторно.

---

### 1.7 Очистка истории имён

**Что делает:**

Удаляет сохранённую историю имён из локального хранилища расширения.

**Как пользоваться:**

Нажми кнопку **Очистить** в блоке **История имён**.

---

## 2. Вкладка «Копия»

### 2.1 Сканирование вопросов

**Что делает:**

Ищет на текущей странице вопрос, тестовое задание или обычное задание и подготавливает текст для копирования.

**Как пользоваться:**

1. Открой страницу с вопросом или заданием.
2. Открой popup.
3. Перейди во вкладку **Копия**.
4. Нажми **Сканировать**.
5. Дождись завершения сканирования.

**Что может найти:**

- обычный вопрос;
- вопрос с одним вариантом ответа;
- вопрос с несколькими вариантами ответа;
- задание с текстовым вводом;
- задание на сопоставление;
- обычное задание без вариантов.

---

### 2.2 Отображение прогресса сканирования

**Что делает:**

Показывает, идёт ли сканирование, сколько времени прошло и сколько вопросов найдено.

**Как пользоваться:**

После нажатия **Сканировать** просто смотри на строку прогресса.

Пример:

```text
Сканирование: 2.5 с / 20.0 с. Найдено: 1
```

---

### 2.3 Скопировать сейчас

**Что делает:**

Копирует найденный вопрос или задание в буфер обмена.

**Как пользоваться:**

1. Сначала нажми **Сканировать**.
2. Когда вопрос найден, нажми **Скопировать сейчас**.
3. Вставь текст куда нужно.

---

### 2.4 Последний результат

**Что делает:**

Показывает последний скопированный или найденный текст вопроса/задания.

**Как пользоваться:**

После сканирования или копирования посмотри поле **Последний результат**.

---

### 2.5 Очистка последнего результата

**Что делает:**

Очищает поле с последним скопированным вопросом.

**Как пользоваться:**

Нажми кнопку **Очистить** возле блока **Последний результат**.

---

### 2.6 Автокопирование

**Что делает:**

Следит за изменениями на странице и автоматически копирует новый вопрос, когда он появляется.

**Как пользоваться:**

1. Нажми **Сканировать**.
2. После успешного сканирования нажми **Включить автокопирование**.
3. Переходи между вопросами на странице.
4. Расширение будет пытаться копировать новый вопрос само.

**Важно:**

Автокопирование работает лучше после первого успешного сканирования, потому что расширение понимает структуру страницы.

---

### 2.7 Скопировать с автопромтом

**Что делает:**

Копирует вопрос вместе с заранее заданной инструкцией для модели.

**Как пользоваться:**

1. Сохрани промт в настройках popup.
2. Перейди во вкладку **Копия**.
3. Нажми **Сканировать**.
4. Нажми **Скопировать с автопромтом**.

**Пример результата:**

```text
Реши задание кратко. Сначала дай ответ, потом объяснение.

Вопрос:
...

Варианты ответа:
1) ...
2) ...
```

---

### 2.8 Скачать логи сканера

**Что делает:**

Скачивает `.txt` файл с техническими логами сканирования вопросов.

**Как пользоваться:**

1. Включи **Режим разработчика** в настройках.
2. Во вкладке **Копия** появится кнопка **Скачать логи сканера .txt**.
3. Нажми её.

**Зачем нужно:**

Для поиска ошибок, если расширение не видит вопрос или неправильно его копирует.

## 3. Вкладка «Оценки»

### 3.1 Посчитать оценки

**Что делает:**

Читает таблицу журнала InternetUrok, собирает оценки по каждому предмету, считает средний балл и предполагаемую итоговую оценку.

**Как пользоваться:**

1. Открой страницу журнала InternetUrok.
2. Открой popup.
3. Перейди во вкладку **Оценки**.
4. Нажми **Посчитать оценки**.

---

### 3.2 Средний балл

**Что делает:**

Считает среднее арифметическое всех найденных оценок по предмету.

**Пример:**

```text
Оценки: 5, 4, 4, 5
Средний балл: 4.5
```

---

### 3.3 Предполагаемая итоговая оценка

**Что делает:**

Переводит средний балл в итоговую оценку.

**Логика:**

```text
4.5 и выше -> 5
3.5 и выше -> 4
2.5 и выше -> 3
ниже 2.5 -> 2
```

---

### 3.4 Список предметов

**Что делает:**

Показывает список предметов, количество оценок, средний балл и итоговую оценку.

**Как пользоваться:**

После нажатия **Посчитать оценки** смотри блок **Список предметов**.

---

## 4. Вкладка «Ответы»

### 4.1 Поле для ответа модели

**Что делает:**

Позволяет вставить ответ модели, чтобы расширение подсветило правильные ответы на странице теста.

**Формат:**

```text
Задание N
Тип: single | multiple | match | input
Ответ: ...
Пояснение: ...
```

---

### 4.2 Подсветить на странице

**Что делает:**

Разбирает вставленный ответ модели и подсвечивает правильные элементы на открытой странице.

**Как пользоваться:**

1. Открой страницу с тестом.
2. Скопируй ответ модели.
3. Вставь его во вкладку **Ответы**.
4. Нажми **Подсветить на странице**.

---

### 4.3 Подсветка single-ответов

**Что делает:**

Подсвечивает один правильный вариант ответа.

**Пример:**

```text
Тип: single
Ответ: 2
```

Будет подсвечен второй вариант.

---

### 4.4 Подсветка multiple-ответов

**Что делает:**

Подсвечивает несколько правильных вариантов ответа.

**Пример:**

```text
Тип: multiple
Ответ: 1, 3, 4
```

Будут подсвечены варианты 1, 3 и 4.

---

### 4.5 Подсветка input-ответов

**Что делает:**

Показывает подсказку для текстового ответа.

**Пример:**

```text
Тип: input
Ответ: таблиц
```

На странице появится заметка с нужным ответом.

---

### 4.6 Подсветка match/drag-ответов цветами

**Что делает:**

Для заданий на сопоставление подсвечивает пару **поле + правильный ответ** одним цветом.

**Пример ответа модели:**

```text
Тип: match
Ответ:
- Name: Alex Brown => Personal details
- Manchester College => Education
```

**Что увидит пользователь:**

- `Name: Alex Brown` и `Personal details` будут одним цветом.
- `Manchester College` и `Education` будут другим цветом.

---

### 4.7 Убрать подсветку

**Что делает:**

Удаляет подсветку ответов, цветные match-пары и заметки с текущей страницы.

**Как пользоваться:**

Нажми **Убрать подсветку** во вкладке **Ответы**.

---

### 4.8 Очистить текст

**Что делает:**

Очищает поле, куда вставляется ответ модели.

**Как пользоваться:**

Нажми **Очистить текст**.

---

## 5. Вкладка «Профиль → Я»

### 5.1 Чтение имени InternetUrok

**Что делает:**

Пытается найти имя пользователя на странице InternetUrok.

**Как пользоваться:**

1. Открой страницу InternetUrok.
2. Открой popup.
3. Перейди в **Профиль → Я**.
4. Данные обновятся автоматически, если расширение сможет их прочитать.

---

### 5.2 Чтение класса

**Что делает:**

Пытается найти класс ученика на странице InternetUrok, например `9А`, `10Б` и т.п.

**Как пользоваться:**

Открой страницу журнала или страницу, где виден фильтр класса, затем перейди в **Профиль → Я**.

---

### 5.3 Сохранение профиля

**Что делает:**

После успешного чтения сохраняет имя, класс, статус и время обновления в локальное хранилище.

**Зачем нужно:**

Чтобы данные профиля были видны даже после закрытия popup.

---

### 5.4 Статус сборки

**Что делает:**

Показывает статус из `manifest.json`, например `default`, `Dev` или другой статус, если он указан.

---

### 5.5 Сводка тестов

**Что делает:**

Показывает краткую статистику по найденным тестам:

```text
Проверено ДЗ: ...
Тестов: ...
Ошибок загрузки: ...
Пропущено с оценкой: ...
```

---

## 6. Вкладка «Профиль → Тесты»

### 6.1 Найти тесты

**Что делает:**

Сканирует журнал InternetUrok, ищет домашние задания, проверяет, какие из них являются тестами, и сохраняет результат.

**Как пользоваться:**

1. Открой журнал InternetUrok.
2. Открой popup.
3. Перейди в **Профиль → Тесты**.
4. Нажми **Найти тесты**.
5. Дождись завершения поиска.

---

### 6.2 Гибкая настройка

**Что делает:**

Открывает дополнительные настройки поиска и подсветки тестов.

**Как пользоваться:**

1. Перейди в **Профиль → Тесты**.
2. Нажми **Гибкая настройка**.
3. Выбери нужные параметры.
4. Сохрани или просто используй выбранный режим, если настройки сохраняются автоматически.

---

### 6.3 Выбор предмета

**Что делает:**

Ограничивает поиск тестов одним предметом.

**Как пользоваться:**

1. Открой **Гибкую настройку**.
2. В поле предмета выбери или введи нужный предмет.
3. Нажми **Найти тесты**.

**Если предмет не выбран:**

Расширение ищет тесты по всем предметам.

---

### 6.4 Выбор недели

**Что делает:**

Ограничивает поиск тестов одной неделей журнала.

**Как пользоваться:**

1. Открой **Гибкую настройку**.
2. Выбери нужную неделю.
3. Нажми **Найти тесты**.

**Если неделя не выбрана:**

Расширение ищет тесты по всем неделям.

---

### 6.5 Сканировать всё сразу

**Что делает:**

Старый режим работы: сканирует все предметы и все недели.

**Как пользоваться:**

Не выбирай предмет и неделю, затем нажми **Найти тесты**.

---

### 6.6 Автоматическая подсветка

**Что делает:**

Автоматически подсвечивает найденные тесты на странице журнала без открытия popup.

**Как пользоваться:**

1. Один раз нажми **Найти тесты**, чтобы сохранить список тестов.
2. Открой **Гибкую настройку**.
3. Включи **Автоматическая подсветка**.
4. Перейди на страницу журнала InternetUrok.
5. Подсветка применится сама.

**Важно:**

Автоподсветка использует уже сохранённый список тестов. Если список пустой, сначала нужно выполнить поиск.

---

### 6.7 Подсветить тесты

**Что делает:**

Подсвечивает найденные тесты в таблице журнала.

**Как пользоваться:**

1. Сначала нажми **Найти тесты** или импортируй список.
2. Открой страницу журнала.
3. Нажми **Подсветить тесты**.

---

### 6.8 Отображение количества вопросов возле ДЗ

**Что делает:**

При подсветке тестов добавляет число вопросов рядом с надписью `ДЗ`.

**Пример:**

```text
ДЗ 10
```

Это значит, что в тесте 10 вопросов.

---

### 6.9 Убрать подсветку тестов

**Что делает:**

Удаляет подсветку тестов, подсветку ячеек и метки с количеством вопросов.

**Как пользоваться:**

Нажми **Убрать подсветку** во вкладке **Профиль → Тесты**.

---

### 6.10 Скопировать список

**Что делает:**

Копирует список найденных тестов в буфер обмена.

**Как пользоваться:**

1. Найди тесты.
2. Нажми **Скопировать список**.
3. Вставь список куда нужно.

---

### 6.11 Скачать список `.txt`

**Что делает:**

Скачивает найденные тесты в текстовый файл.

**Как пользоваться:**

Нажми **Скачать список .txt**.

---

### 6.12 Загрузить чужой список `.txt`

**Что делает:**

Импортирует список тестов из текстового файла.

**Как пользоваться:**

1. Нажми **Загрузить чужой список .txt**.
2. Выбери файл.
3. После импорта тесты появятся в списке.

---

### 6.13 Удаление теста из списка

**Что делает:**

Удаляет конкретный тест из сохранённого registry.

**Как пользоваться:**

В списке тестов нажми кнопку удаления у нужного теста, если она отображается в текущем виде списка.

---

### 6.14 Вид списка тестов

**Что делает:**

Позволяет переключить отображение найденных тестов.

**Режимы:**

- **Новый** — карточки с более современным видом.
- **Старый** — классический вид списка.

**Как пользоваться:**

Во вкладке **Профиль → Тесты** выбери нужный вариант в блоке **Вид списка тестов**.

---

### 6.15 Группировка по четвертям

**Что делает:**

Раскладывает найденные тесты по четвертям: 1, 2, 3, 4.

**Зачем нужно:**

Чтобы тесты из разных четвертей не смешивались.

---

### 6.16 Скачать логи поиска тестов

**Что делает:**

Скачивает `.txt` файл с логами поиска тестов.

**Как пользоваться:**

1. Включи **Режим разработчика**.
2. Перейди в **Профиль → Тесты**.
3. Нажми **Скачать логи поиска тестов .txt**.

---

## 7. Настройки popup

### 7.1 Темы оформления

**Что делает:**

Меняет цветовую тему popup.

**Доступные темы:**

```text
dark, light, sunset, mint, ocean, rose, midnight, neon, aurora, graphite, candy, forest, royal, matrix, ice, lava
```

**Как пользоваться:**

1. Нажми шестерёнку в popup.
2. Выбери тему.

### 7.2 Режимы UI

**Что делает:**

Меняет стиль и плотность интерфейса popup.

**Примеры режимов:**

```text
new, compact, focus, premium, dock, cards, split, old
```

**Как пользоваться:**

1. Нажми шестерёнку.
2. В блоке **Режим UI** выбери нужный режим.

---

### 7.3 Цвет подсветки

**Что делает:**

Позволяет выбрать цвет подсветки ответов и тестов.

**Режимы:**

- **От темы** — цвет берётся из выбранной темы.
- **Свой цвет** — можно указать HEX или RGB.

**Как пользоваться:**

1. Нажми шестерёнку.
2. В блоке **Подсветка ответов** выбери режим.
3. Если выбран свой цвет, введи `#22c55e` или `rgb(34, 197, 94)`.
4. Нажми **Сохранить цвет**.

---

### 7.4 Сброс цвета подсветки

**Что делает:**

Возвращает цвет подсветки к стандартному режиму.

**Как пользоваться:**

Нажми **Сбросить** в настройках цвета подсветки.

### 7.5 Режим разработчика

**Что делает:**

Показывает дополнительные кнопки логов.

**Как пользоваться:**

### 7.6 Пользовательский промт

4. Потом используй **Скопировать с автопромтом** во вкладке **Копия**.

### 7.7 Очистить промт

## 8. Бренд, версия и ping

### 8.1 Отображение версии

**Что делает:**

Показывает текущую версию расширения из `manifest.json`.

**Как пользоваться:**

Версия отображается в шапке popup рядом с названием **God Game**.

---

### 8.2 Проверка актуальности версии

**Что делает:**

Сравнивает текущую версию с версией в GitHub manifest.

**Зачем нужно:**

Чтобы понимать, актуальная ли установлена версия расширения.

---

### 8.3 Ping-индикатор

**Что делает:**

Проверяет доступность сайта/сервиса и показывает состояние ping в шапке popup.

**Как пользоваться:**

Открой popup и посмотри строку `ping` рядом с информацией об авторе.

---

## 9. Служебные функции расширения

### 9.1 Usage ping

**Что делает:**

Регистрирует использование расширения и увеличивает счётчик запусков/пользователей.

**Когда срабатывает:**

- при открытии popup;
- при загрузке content script на странице.

---

### 9.2 Service worker

**Что делает:**

Работает в фоне и принимает сообщения от popup/content script.

**Основные задачи:**

- выполнить usage ping;
- проверить доступность сайта;
- сохранить результат в `chrome.storage.local`.

---

### 9.3 Content script

**Что делает:**

Работает внутри страниц и выполняет действия, связанные с DOM:

- поиск вопросов;
- сканирование журнала;
- подсветка тестов;
- подсветка ответов;
- чтение профиля InternetUrok;
- автоподсветка.

---

### 9.4 Injected script

**Что делает:**

Внедряется прямо в страницу, чтобы получить доступ к объектам страницы, например `window.APP` в Jitsi.

**Зачем нужен:**

Без него обычный content script не сможет менять имя в Jitsi через внутренний store страницы.

---

### 9.5 Хранение данных

**Что делает:**

Сохраняет настройки и данные расширения в `chrome.storage.local`.

**Что хранится:**

```text
тема
режим UI
цвет подсветки
пользовательский промт
история имён
последний скопированный вопрос
список найденных тестов
данные профиля
режим разработчика
вид списка тестов
гибкие настройки тестов
состояние автоподсветки
usage ping state
```

---

## 10. Короткий сценарий использования

### Найти и подсветить тесты

1. Открой журнал InternetUrok.
2. Открой popup.
3. Перейди в **Профиль → Тесты**.
4. При необходимости открой **Гибкая настройка** и выбери предмет/неделю.
5. Нажми **Найти тесты**.
6. Нажми **Подсветить тесты**.
7. Если хочешь, включи **Автоматическая подсветка**, чтобы дальше подсветка работала сама.

### Скопировать вопрос

1. Открой страницу с заданием.
2. Открой popup.
3. Перейди в **Копия**.
4. Нажми **Сканировать**.
5. Нажми **Скопировать сейчас** или **Скопировать с автопромтом**.

### Подсветить ответы модели

1. Открой страницу теста.
2. Получи ответ модели в нужном формате.
3. Открой popup.
4. Перейди в **Ответы**.
5. Вставь ответ.
6. Нажми **Подсветить на странице**.

### Посчитать оценки

1. Открой журнал InternetUrok.
2. Открой popup.
3. Перейди в **Оценки**.
4. Нажми **Посчитать оценки**.

### Сменить имя в Jitsi

1. Открой Jitsi.
2. Открой popup.
3. Перейди в **Имя**.
4. Введи новое имя.
5. Нажми **Применить имя**.
